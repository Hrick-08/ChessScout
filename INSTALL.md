# Installation Guide

Follow these steps to install **Chess Scout** in your browser directly from the source code.

## 1. Download the Extension

You can get the code in two ways:

### Option A: Download as ZIP (Easiest)
1. Click the green **Code** button at the top of this GitHub repository.
2. Select **Download ZIP**.
3. Locate the downloaded file on your computer and **Extract/Unzip** it. You should now have a folder named `ChessScout-main` (or similar).

### Option B: Clone via Git
If you have Git installed, run:
```bash
git clone https://github.com/Hrick-08/ChessScout.git
```

---

## 2. Load into Chrome (or Edge/Brave)

1. Open your browser and go to the Extensions page:
   - **Chrome:** `chrome://extensions`
   - **Edge:** `edge://extensions`
   - **Brave:** `brave://extensions`
2. In the top-right corner, toggle **Developer mode** to **ON**.
3. Click the **Load unpacked** button that appears in the top-left.
4. In the file picker, select the folder where you extracted/cloned the code.
   - *Note: Select the root folder that contains the `manifest.json` file.*
5. The **Chess Scout** extension should now appear in your list of extensions!

---

## 3. Start Scouting!

1. Go to [chess.com](https://www.chess.com).
2. Start a new game (Rapid, Blitz, or Bullet).
3. As soon as the game starts and your opponent is detected, the **Chess Scout** icon will appear next to the opponent's username!

---

## Troubleshooting

- **Icon not appearing?**
  - Refresh the page once the game has started.
  - Make sure "Developer mode" is still ON.
  - Check that the extension is enabled in your browser settings.
- **Updating the extension:**
  - If you download a new version, just click the **Reload** icon (circular arrow) on the extension card in `chrome://extensions`.

---

## Privacy Note
Chess Scout only uses the **Chess.com Public API**. It does not require your login credentials and does not track your personal data.
